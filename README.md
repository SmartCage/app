# Smart Cage

## About the project
This application helps the parrot owners automate the daily upkeeping of the cages, being able to record sensor data and adjust the environment so it suits the birds.

## Technologies used
<ol>
<li> Python </li>
<li> Flask </li>
<li> MySQL </li>
<li> RestAPI </li>
<li> MQTT API </li>
<li> Swagger </li>
<li> RESTler </li>
<li> Async API </li>
</ol>
